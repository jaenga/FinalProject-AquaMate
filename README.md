AquaMate는 사용자의 일일 수분 섭취량을 기록하고, 목표치를 설정하며, 시간별/주간 섭취 통계를 제공하는 Android 기반 앱입니다.  
기말 프로젝트로 제작되었습니다.

## 📱 주요 기능

- 닉네임 및 목표 수분량 입력 (SharedPreferences 저장)
- 물 마시기 버튼 클릭 시 섭취량 기록
- 시간별/주간 섭취량 통계 시각화 (MPAndroidChart)
- 간단한 온보딩 플로우 및 다중 액티비티 구성

## 🛠 사용 기술

- Kotlin
- Android Studio
- MPAndroidChart
- ViewPager2
- SharedPreferences
